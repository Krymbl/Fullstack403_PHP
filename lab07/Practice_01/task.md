# Практика
Загрузка конфигурации из .env с обработкой ошибок
## Необходимо:

- Установить пакет vlucas/phpdotenv (или другой с обработкой .env)
- Реализовать безопасную загрузку конфигурации с обработкой ошибок
- Создать свои классы исключений
- Логировать все пойманные ошибки

Рекомендуется сразу сделать класс загрузки конфигурации App\Core\Config. Ошибки загрузки конфигурации - это, например, отсутствие файла конфига или отсутствие в нём обязательных параметров.

Пример файла .env:
```
APP_NAME="Мой проект"
APP_ENV=dev
APP_DEBUG=true

DB_HOST=localhost
DB_NAME=myapp
DB_USER=root
DB_PASS=secret
```
Место логов в структуре проекта - /runtime/logs
