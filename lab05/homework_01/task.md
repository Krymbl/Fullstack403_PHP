Домашнее задание
---
**Мини-проект: «Корзина товаров»**

- Разработайте простую корзину покупок с использованием ООП. Данные о товарах в корзине должны храниться в сессии. Можно делать в рамках семестрового проекта, если ваша тема подразумевает какую-то форму магазина. По согласованию, можно заменить задание на аналогичное по сложности, соответствующее вашей теме.

Требования:
---
1. Класс Product (в файле Product):


- Приватные свойства: id, title, price.
- Конструктор.
- Геттеры для всех свойств.
---
2. Класс Cart (в файле Cart):

- Приватное свойство $items — массив, где ключ — id товара, а значение — объект товара или просто id и количество. (Выберите удобную структуру).
- Метод add(Product $product, int $quantity = 1) — добавляет товар в корзину или увеличивает количество.
- Метод remove(int $productId) — удаляет товар из корзины.
- Метод getItems() — возвращает все товары в корзине.
- Метод getTotal() — возвращает общую стоимость корзины.
- Метод clear() — очищает корзину.
---
3. Маршрут index (главная страница):
- Создайте несколько объектов Product (массив товаров).
- Выведите список товаров. Рядом с каждым товаром должна быть кнопка «Добавить в корзину», которая ведет на add?id=ТОВАР_ID.
- Внизу страницы должна быть ссылка «Перейти в корзину» на cart.
---

4. Маршрут add:

- Получает id товара из $_GET.
- Находит товар по id в массиве товаров.
- Запускает сессию (session_start()).
- Создает или извлекает объект Cart из сессии (например, $_SESSION['cart']).
- Вызывает метод add() у объекта корзины, передавая ему товар.
- Перенаправляет пользователя обратно на index (редирект).
---
5. Маршрут cart:

- Запускает сессию, извлекает объект Cart из $_SESSION.
- Выводит список товаров в корзине (используя метод getItems()), их количество, цену за единицу и общую стоимость.
- Рядом с каждым товаром должна быть кнопка «Удалить», ведущая на remove?id=ТОВАР_ID.
- Кнопка «Очистить корзину», ведущая на clear.
---
6. Маршруты remove и clear:

- Реализуют логику удаления одного товара или очистки всей корзины соответственно, используя методы класса Cart.
- После выполнения действия делают редирект на cart.
