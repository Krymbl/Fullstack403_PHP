# Проект - Онлайн магазин товаров

## 1 этап: Каркас приложения с контроллерами

Домашнее задание (lab06):

Требования
* Структура проекта:
    * папка public со всей статикой, включая точку входа index.php, внутри подпапки для css, js и изображений
    * папка src для всего кода
    * файл autoload.php в корне проекта, в котором должны быть описаны все правила автозагрузки (по PSR-4)
* Внутри src
    * отдельные классы для роутера, описывающего маршрутизацию (сами маршруты нужно вынести в отдельный файл: либо статический конфиг, обрабатываемый в index, либо в routes.php, который будет сам передавать роутеру маршруты) и базовый абстрактный контроллер, от которого будут наследоваться остальные контроллеры
    * папка со всеми контроллерами, каждый адрес должен обрабатывать отдельный класс, допустимо обрабатывать логически связанные страницы разными методами одного контроллера
* Весь общий функционал контроллеров должен быть вынесен в абстрактный контроллер, весь функционал запросов (извлечение параметров, определение http метода и т.д.) - в роутер.
  

# Lab06 — Каркас приложения с контроллерами

## Структура проекта

```
project/
│
├── public/                             # Статика и точка входа
│   ├── index.php                       # Точка входа — подключает autoload и routes
│   ├── css/                            # Стили
│   ├── js/                             # Скрипты
│   └── images/                         # Изображения
│
├── src/                                # Весь исходный код
│   │
│   ├── Core/                           # Ядро приложения
│   │   ├── Router.php                  # Маршрутизация, HTTP-методы, параметры запроса
│   │   └── AbstractController.php      # Базовый абстрактный контроллер
│   │
│   ├── Controllers/                    # Конкретные контроллеры
│   │   │
│   │   ├── Admin/                      # Контроллеры админ-панели
│   │   │   ├── CatalogController.php   # Просмотр всех товаров
│   │   │   ├── AddController.php       # Добавление товара
│   │   │   └── EditController.php      # Редактирование / удаление товара
│   │   │
│   │   └── User/                       # Контроллеры пользовательской части
│   │       ├── AuthController.php      # Вход, регистрация, выход
│   │       ├── HomeController.php      # Главная — акции, баннер, хиты
│   │       ├── CatalogController.php   # Каталог с фильтрацией и пагинацией
│   │       ├── ProductController.php   # Страничка одного товара
│   │       ├── CartController.php      # Корзина
│   │       ├── OrderController.php     # Оформление заказа + успех / ошибка
│   │       └── ProfileController.php   # Личный кабинет + история заказов
│   │
│   ├── Models/                         # Модели — сущности базы данных
│   │   ├── User.php                    # Пользователь, роль, сессия
│   │   ├── Product.php                 # Товар, количество, доступность
│   │   ├── Cart.php                    # Корзина, связь с пользователем
│   │   ├── Order.php                   # Заказ, данные получателя
│   │   └── OrderItem.php              # Позиция заказа — товар + цена на момент покупки
│   │
│   └── routes.php                      # Список маршрутов → передаёт их роутеру
│
├── ErrorController.php                 # Обработка ошибок 404, 403
└── autoload.php                        # PSR-4 автозагрузка всех классов из src/
```

---

## Зоны ответственности

| Файл / Папка | Ответственность |
|---|---|
| `public/index.php` | Точка входа: подключает `autoload.php` и `routes.php`, запускает роутер |
| `autoload.php` | PSR-4 автозагрузка всех классов из `src/` |
| `src/Core/Router.php` | Маршрутизация, определение HTTP-метода, извлечение параметров запроса |
| `src/Core/AbstractController.php` | Общий функционал всех контроллеров (рендер, редирект и т.д.) |
| `src/Controllers/Admin/` | Контроллеры админ-панели |
| `src/Controllers/User/` | Контроллеры пользовательской части |
| `src/Models/` | Модели сущностей базы данных |
| `src/routes.php` | Декларация маршрутов, передача их в роутер |
| `ErrorController.php` | Обработка ошибок 404, 403 |
