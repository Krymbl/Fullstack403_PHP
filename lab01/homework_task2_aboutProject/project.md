**Онлайн-магазин с товарами** 
-
**Странички:**
* Авторизация
* Каталог с фильтрацией
* Страничка товара
* Корзина
* Оформление заказа (форма для сбора данных получателя)
* Страница успеха / ошибки

**Странички администрирования:**
 * Просмотр всех товаров
 * Изменение данных товара
 * Добавление товара
---
**Как устроено?**
-
В каталоге товаров будут отображаться данные из базы данных/json. Корзина доступна аторизованным пользователям. Пользователи хранятся в базе данных/json. Нынешний пользователь будет храниться в сессии, чтобы при обновлении странички состояние сохранилось. Пользователь связан с корзиной по внешнему ключу. Корзины хранится в базе данных. Нельзя добавить в корзину товар, который недоступен (кол-во товара в базе - 0). При добавлении товара в корзину и оформлении заказа еще раз происходит проверка наличия товара.

У пользователей с ролью админ смогут изменять каталог товаров (добавлять товар, удалять, изменять кол-во). Отдельная страничка администрирования.

После заказа товаров их кол-во  измениться
